🌲 Forest Trials: The Triple Adventure

Forest Trials is a dynamic adventure game developed in Unity that seamlessly blends three distinct gameplay mechanics into one cohesive experience. Each level introduces new challenges, unique atmospheres, and custom soundtracks.

🎮 Gameplay & Level Design
The game features a progressive three-stage journey, each with its own mechanics and mood:

Level 1: The Gathering 🍎
Mechanic: Arcade item collection.
Objective: Collect falling food items to gain strength before the journey begins. A test of reaction and precision.

Level 2: Enchanted Woods 💎
Mechanic: 2D Platforming.
Objective: Navigate through a lush forest by jumping across platforms. Collect magical crystals and mushrooms while avoiding environmental hazards.

Level 3: The Goblin Showdown ⚔️
Mechanic: Boss Fight Logic.
Objective: The ultimate test! Use your skills to defeat the cunning Goblin Boss in a final combat encounter.

🎵 Audio Experience
To enhance immersion, each level features a unique music track specifically chosen to match the intensity and theme of the environment, from peaceful gathering to an epic boss battle.

🛠 Tech Stack
Engine: Unity 2022.3 LTS
Language: C#
Physics: 2D Physics Engine (Colliders, Triggers, Rigidbody2D)
Version Control: Git / GitHub
